Humor com as IAs Generativas

## 📒 Descrição
Criar conteúdo com IAs Generativas, combinando texto, áudio e vídeo.

## 🤖 Tecnologias Utilizadas
- ChatGPT
- Microsft Copilot
- D-ID

## 🧐 Processo de Criação
Inspirado numa piada antiga do saudoso Chico Anysio, ainda nos tempos do rádio,
em que dialogava com um garçom num restaurante e todas as palavras que dizia,
iniciava com a letra "F", resolvi criar um vídeo com uma piada neste estilo.

No MS-Copilot pedi que criasse uma imagem de um homem, um humorista, de pé, olhando de frente, com as mãos ao lado do corpo.
Então pedi para o ChatGPT criar uma piada de stand up com todas palavras iniciando com a letra "F".

Tive que refinar, pois ele inseriu algumas poucas palavras que não iniciavam com a letra "F",
mas a segunda versão ficou quase perfeita, então copiei o texto e  fiz pequenos ajustes diretamente no D-ID. 

Usei o D-ID, para criar o projeto de vídeo, com o avatar baseado na imagem que criei no MS-Copilot,
mas pedi para remover o fundo, que era uma parede de tijolos e depois eu inseri uma imagem de fundo do próprio D-ID, uma cozinha.

Enfim, escolhi uma voz padrão multilingual do D-ID e gerei o vídeo.
